# Ericka Barbosa — Nutricionista

Static website (HTML/CSS) for Ericka Barbosa, nutricionista brasileira na Itália.
Multi-page, responsive, SEO/AEO optimized. No build step — pure static files.

## Deploy (Vercel)
1. Push this repo to GitHub.
2. Vercel → Add New… → Project → Import this repository.
3. Framework preset: **Other**. Root directory: `./`. Build command: none. Output: `./`.
4. Deploy. Vercel serves the static files directly.

## Structure
- `index.html`, `sobre.html`, `metodo.html`, `planos.html`, `resultados.html`, `faq.html`, `contato.html` + legal pages
- `css/main.css` — shared styles
- `img/` — logo, photos, results images, icons, share image
- `robots.txt`, `sitemap.xml`, `site.webmanifest`
